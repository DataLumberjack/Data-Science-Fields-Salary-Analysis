# Data Science Salary Analysis

## Overview
This project explores salary distributions and job categories within the data and analytics job market.

## Dataset
Data Science Fields Salary Categorization dataset.

## Methodology
- Data cleaning and preparation
- Exploratory analysis
- Salary segmentation by role and category

## Tools & Technologies
- Python
- Pandas
- SQL

## Key Takeaways
The project provides insights into job market trends and demonstrates analytical storytelling.
